# AquaLink

## Overview

AquaLink is a mobile application for monitoring coral reef health and ocean conditions using smart buoy networks and satellite data. The platform enables marine conservationists and researchers to track reef bleaching events, sea surface temperatures, and climate impacts on marine ecosystems in real-time.

## Features

- **Real-time Buoy Data**: Live sea surface temperature and weather conditions
- **Coral Health Monitoring**: Bleaching event alerts and tracking
- **Historical Analytics**: Long-term climate trend visualization
- **Offline Capability**: Data synchronization for remote field work
- **Multi-region Support**: Global reef monitoring network
